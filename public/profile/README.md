# Denny Profile Photo Directory

Place your profile photo in this folder to display it on the interactive lanyard card.

## Supported File Names & Formats:

The portfolio system dynamically scans for the following formats in order, falling back to a premium vector coding avatar if none are found:

1. `avatar.webp`
2. `avatar.png`
3. `avatar.jpg`
4. `avatar.jpeg`

Simply rename your profile picture to `avatar` with one of the extensions above and drop it directly into this directory (`/public/profile/`).
