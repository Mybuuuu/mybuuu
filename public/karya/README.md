# Direktori Karya Portofolio (Portfolio Artworks)

Tempatkan file karya desain dan proyek Anda di dalam folder ini untuk menampilkan karya Anda secara dinamis pada halaman portofolio.

## 📁 Panduan Upload File Karya:

Untuk mengganti gambar ilustrasi/mockup di halaman **Desain Visual (Creative Design)**, ganti file default dengan karya Anda sendiri menggunakan nama dan format berikut:

1. **Desain Komunitas Gereja (Church Youth Design)**
   - **Nama File:** `church_media.png`
   - **Ukuran Rekomendasi:** 800 x 600 piksel (Rasio 4:3) atau 1200 x 900 piksel.
   - **Fungsi:** Ditampilkan di dalam card pengalaman pertama di bagian "Creative Visual Design".

2. **Promosi Es Teh Poci (Es Teh Poci Branding)**
   - **Nama File:** `estehpoci_media.png`
   - **Ukuran Rekomendasi:** 800 x 600 piksel (Rasio 4:3) atau 1200 x 900 piksel.
   - **Fungsi:** Ditampilkan di dalam card pengalaman kedua di bagian "Visual Promotional Design".

---

## 💡 Tips untuk Hasil Terbaik (UI/UX):
- Pastikan ekstensi file adalah `.png` (huruf kecil). Jika Anda memiliki file `.jpg` atau `.webp`, silakan konversi ke `.png` atau ganti ekstensi filenya secara tepat.
- Gunakan kompresi gambar (seperti TinyPNG) sebelum mengunggah agar halaman website dapat dimuat dengan sangat cepat di perangkat mobile.
